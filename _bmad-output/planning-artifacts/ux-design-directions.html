Resume Enhancer — UX Design Directions

6 design directions for the core editor experience. Evaluate layout, information hierarchy, and interaction style.

As you explore, consider: Which layout feels most intuitive for the editor + chat combination? Which visual weight matches the "serious tool" aesthetic? Which navigation pattern keeps resume state clearest?

D1: Command Center

Three-column peer layout. Editor and chat are equal partners. Global nav on left. Cursor/Linear aesthetic — dense, keyboard-first, professional.

Layout: 3-column, equal-weight editor + chat
Nav: Icon sidebar (left)
Density: Medium-dense
Best for: Power users, keyboard-first flow
